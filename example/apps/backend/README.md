# Forge Example Backend

This is a Go API server that uses code generated by Forge.

## Development

### Prerequisites

- Go 1.21+
- MongoDB running locally or via Docker

### Running locally

```bash
# Set environment variables
export MONGO_URI=mongodb://localhost:27017
export DB_NAME=forge_example
export PORT=8080

# Run the server
go run .
```

### Running with Docker

```bash
docker build -t forge-backend .
docker run -p 8080:8080 -e MONGO_URI=mongodb://host.docker.internal:27017 forge-backend
```

## API Endpoints

The server exposes the following endpoints (generated by Forge):

- `GET /tasks/` - List all tasks
- `POST /tasks/` - Create a new task
- `GET /tasks/{id}` - Get a task by ID
- `PUT /tasks/{id}` - Update a task
- `DELETE /tasks/{id}` - Delete a task

- `GET /projects/` - List all projects
- `POST /projects/` - Create a new project
- `GET /projects/{id}` - Get a project by ID
- `PUT /projects/{id}` - Update a project
- `DELETE /projects/{id}` - Delete a project

- `GET /events/` - List all events
- `POST /events/` - Create a new event
- `GET /events/{id}` - Get an event by ID

## Configuration

| Environment Variable | Description            | Default                     |
| -------------------- | ---------------------- | --------------------------- |
| `MONGO_URI`          | MongoDB connection URI | `mongodb://localhost:27017` |
| `DB_NAME`            | Database name          | `forge_example`             |
| `PORT`               | Server port            | `8080`                      |
